# NAT
Står for Network Address Translation. Oversetter èn eller flere [[02 IPv4-adresser|IP-adresser]] til èn IP-adresse.

I et vanlig nettverk kan man ha ulike interne ip-adresser, men kun èn offentlig ip-adresse.
